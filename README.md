## 股票预警系统(Stock alert system)  


​    
​    
### 项目简介  

股票预警系统(Stock alert system)致力于为个人投资者提供实时股票监控，当股价达到合理位置自动预警。  

​     

### 项目运行环境  

- JDK 1.8+  
- MySQL 5.7+  
- Maven 3+  

​    

### 项目技术栈  

- Spring boot 2.4.2  
- Mybatis plus 3.4.2  
- Swagger 3.0 / OpenApi 3.0  

​    

### 运行  

#### 可执行文件(Linux/Unix)  

下载项目打包文件  

解压  

修改数据库信息  

修改邮箱信息  

执行启动脚本  

停止项目  



#### 源码运行(Windows/Linux/Unix)  

克隆项目  

修改配置信息(数据库、邮箱等)  

maven 打包  

启动项目
